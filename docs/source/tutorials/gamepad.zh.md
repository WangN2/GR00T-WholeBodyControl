# 使用手柄进行动作追踪与运动学规划

使用 Unitree 无线手柄控制机器人播放参考动作并运行基于规划器的 locomotion（使用 `--input-type gamepad`）。

```{admonition} 前置条件
:class: note
在进行下一步之前，请先完成[安装指南](../getting_started/installation_deploy)并构建项目。
```

```{warning}
手柄接口需要将物理 Unitree 无线手柄连接到机器人。它在 **sim2sim 中不可用** —— 请先在仿真中使用[键盘接口](keyboard.md)进行测试。
```

```{admonition} 紧急停止
:class: danger
随时按 **Select** 即可立即停止控制并退出。请始终保持一只手准备按 **Select**。
```

## 启动

```bash
# 在 gear_sonic_deploy/ 目录下
bash deploy.sh --input-type gamepad real
```

## 分步指南：普通模式（参考动作追踪）

普通模式播放预加载的参考动作。这是程序启动时的默认模式。

1. 按 **Start** 启动控制系统。
2. 按 **A** 播放当前参考动作 —— 机器人将执行到完成。
3. 按 **R1** 切换到下一个动作序列，或按 **L1** 切换到上一个。
4. 再次按 **A** 播放新动作。
5. 要在动作中途停止并返回第一帧，按 **B** —— 机器人暂停但不终止策略。
6. 使用 **十字键左 / 右** 微调朝向（每次 ±0.1 rad）。
7. 按 **X** 或 **Y** 重新初始化基座四元数并将朝向重置为零，即机器人会将当前朝向视为参考动作第一帧的朝向。
8. 完成后，按 **Select** 停止控制并退出。

## 分步指南：规划器模式（实时动作生成）

规划器模式提供摇杆控制 —— 用左摇杆控制移动方向，右摇杆控制朝向，并可在不同 locomotion 模式间切换。

1. 在普通模式下，按 **F1** 切换到规划器模式。终端将打印 `Planner enabled`。
2. 机器人默认进入慢走模式（模式 1）。推动**左摇杆向前**即可行走 —— 移动方向根据摇杆角度相对于当前朝向计算。
3. 左右推动**右摇杆**以平滑旋转朝向（连续，每帧 ±0.02 rad）。
4. 按 **R1** 切换到下一个移动模式（Idle → Slow Walk → Walk → Run → Squat → Kneel Two Legs → Kneel → Idle → …）。按 **L1** 反向切换。
5. 按住 **R2** 增加速度（站立模式 1–3）或高度（下蹲模式 4–6）。按住 **L2** 减小。按住时速度/高度每帧变化 ±0.02。
6. 当左摇杆在死区时，站立模式自动切换为 Idle；下蹲/跪姿模式保持速度为 0 的姿势。
7. 要暂停，按 **B** —— 机器人立即重置为 Idle。
8. 再次按 **F1** 返回普通模式，或按 **Select** 停止并退出。

## 控制参考

### 系统控制（两种模式通用）

| 按键 | 动作 |
|--------|--------|
| **Start** | 启动控制系统 |
| **Select** | 停止控制并退出（紧急停止） |
| **F1** | 在普通 / 规划器模式间切换 |
| **X** 或 **Y** | 重新初始化基座四元数并重置朝向 |
| **十字键左 / 右** | 调整 delta 朝向（±0.1 rad） |

### 普通模式按键

| 按键 | 动作 |
|--------|--------|
| **A** | 播放当前动作到完成 |
| **B** | 从开头重启当前动作（暂停在第 0 帧） |
| **L1** / **R1** | 上一个 / 下一个动作序列 |

### 规划器模式按键

**移动：**

| 输入 | 动作 |
|-------|--------|
| **左摇杆** | 移动方向（根据摇杆角度 + 当前朝向计算） |
| **右摇杆** | 朝向（连续旋转，±0.02 rad/帧） |

**模式与速度：**

| 按键 | 动作 |
|--------|--------|
| **L1** / **R1** | 上一个 / 下一个移动模式（循环 0–6） |
| **L2**（按住） | 减小速度（模式 1–3）或高度（模式 4–6），±0.02/帧 |
| **R2**（按住） | 增加速度（模式 1–3）或高度（模式 4–6），±0.02/帧 |
| **A** | 播放 / 恢复动作 |

**紧急：**

| 按键 | 动作 |
|--------|--------|
| **B** | 暂停（重置为 Idle） |
| **Select** | 停止控制并退出 |

## 移动模式

使用 **L1** / **R1** 在 7 个模式间循环。使用 **L2** / **R2**（按住）调整速度或高度。

| ID | 模式 | L2/R2 调整项 | 范围 |
|----|------|---------------|-------|
| 0 | Idle | — | — |
| 1 | Slow Walk | 速度 | 0.2–0.8 m/s |
| 2 | Walk | 速度 | 0.8–1.5 m/s |
| 3 | Run | 速度 | 1.5–3.0 m/s |
| 4 | Squat | 高度 | 0.1–0.8 m |
| 5 | Kneel (two legs) |  — |  — |
| 6 | Kneel |  — |  — |

```{tip}
对于横向（侧步）移动，建议将目标速度保持在约 **0.4 m/s**。侧步时速度过高可能导致机器人双脚碰撞，因为横向步态需要交叉摆腿。
```

```{note}
从相邻模式进入 Squat（模式 4）时，高度会自动初始化为 0.8 m。
```
